# 多语言支持说明

## 功能特点

应用现已支持中英文双语切换：

- **中文 (zh)** - 默认语言
- **English (en)** - 英语

## 使用方法

1. 访问应用时，默认显示中文界面：`http://localhost:3001/zh`
2. 也可以直接访问英文界面：`http://localhost:3001/en`
3. 在页面右上角点击语言切换按钮（中文/English）即可切换语言

## 技术实现

使用 `next-intl` 库实现国际化：

### 文件结构

```
messages/
  ├── zh.json          # 中文翻译
  └── en.json          # 英文翻译

src/
  ├── i18n/
  │   ├── request.ts   # i18n 配置
  │   └── routing.ts   # 路由配置
  ├── components/
  │   └── LanguageSwitcher.tsx  # 语言切换组件
  ├── middleware.ts    # 国际化中间件
  └── app/
      ├── [locale]/    # 语言动态路由
      │   ├── layout.tsx
      │   └── page.tsx
      └── page.tsx     # 根页面重定向
```

### 添加新语言

1. 在 `messages/` 目录下创建新的语言文件（如 `ja.json`）
2. 在 `src/i18n/routing.ts` 中添加新语言代码：
   ```typescript
   locales: ['zh', 'en', 'ja']
   ```
3. 在 `LanguageSwitcher.tsx` 中添加新语言按钮

### 添加新的翻译文本

在 `messages/zh.json` 和 `messages/en.json` 中添加对应的翻译键值对，然后在组件中使用：

```typescript
const t = useTranslations();
<p>{t('your.translation.key')}</p>
```

## 路由说明

- `/` - 自动重定向到 `/zh`
- `/zh` - 中文版本
- `/en` - 英文版本
- `/[locale]/api/*` - API 路由（语言无关）
